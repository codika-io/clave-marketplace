# Clave Marketplace

Plugin marketplace for the [Clave](https://github.com/codika-io/clave) desktop app — a multi-session Claude Code terminal manager.

## Plugins

| Plugin | Description |
|--------|-------------|
| **workspace-builder** | Create and configure `.clave` workspace files with groups, sessions, terminals, icons, and toolbar actions |

## Installation

Add this marketplace to your Claude Code settings:

```json
{
  "plugins": ["codika-io/clave-marketplace"]
}
```
